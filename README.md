# TBLTECHNERDS-final-exam
test for a nodejs position

# TO RUN
1. clone the repository
2. run `npm install`
3. `cd server`
4. `npm run dev`
5. Go to localhost:3333 on your browser

note: the database runs on mongodb atlas, you can change the uri in `.env` or just use mine.
